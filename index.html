The Trust Deficit
in Real Estate

The Indian rental market is chaotic. Relying on WhatsApp, cash transactions, and unverified brokers creates extreme liability. Our market analysis of Jaipur reveals critical pain points:

The Brokerage Tax

Students pay 1-2 months' rent as brokerage fees for zero post-move-in support.

Cultural Friction

Traditional platforms lack filters for "Veg/Non-Veg" or "Bachelor Friendly", leading to rejection at the doorstep.

Legal Vulnerability

Verbal agreements bypass the Model Tenancy Act, risking security deposits.

Organized Unorganized

Market Efficiency Gap

Technical Architecture

For the prototype phase, we architected a "Serverless-Simulation" model to demonstrate full functionality with zero latency.

Frontend Logic

  • React 19 & TypeScript

    Strict typing interfaces for `Property` and `User` models to prevent runtime errors.

  • Context API

    Global state management for Authentication (JWT storage) and Theme preferences.

  • Framer Motion

    GPU-accelerated animations for page transitions and modal popups.

AI & Services

  • Google Gemini API

    Model: gemini-3-flash. Generates property descriptions and handles natural language search queries.

  • Mock Service Layer

    A custom TypeScript service that intercepts API calls and routes them to localStorage, simulating a Node.js backend.
